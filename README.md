# Quest Interactive Website

Production-ready marketing website for Quest Interactive, a B2B AI automation agency serving SMEs and growing businesses.

## Tech stack
- Next.js 16 (App Router)
- React 19
- TypeScript (strict mode)
- Tailwind CSS 4

## Pages
- /
- /how-it-works
- /features
- /case-studies
- /pricing
- /contact

## Brand positioning included
- Headline and messaging focused on speed + 24/7 availability
- SME-focused value proposition and conversion-oriented CTAs
- Placeholder proof points and case studies formatted for easy replacement with real results

## Local development
```bash
npm install
npm run dev
```

## Quality checks
```bash
npm run lint
npm run build
```

## Deployment
This project is configured for GitHub + Vercel deployment.
- Push to `main` for source control
- Deploy to production via Vercel CLI or linked Vercel project

## Notes
- Contact details and WhatsApp link currently use placeholders in `src/lib/site-data.ts`
- Pricing values are placeholders and should be updated before launch
